# Disc Golf Disc Cleaner

An automated disc cleaning station designed to thoroughly clean disc golf discs after muddy rounds. This project was developed as part of Olin College of Engineering's Principles of Engineering course by a team of 5 students.

## About This Website

This is a React-based project portfolio website showcasing the complete development journey of the Disc Golf Disc Cleaner. The website includes:

- **Home** - Project overview, key features, dream vision, and MVP details
- **Team & Reflections** - Team member profiles with individual learning goals and reflections
- **System Design** - Detailed mechanical, electrical, and software architecture
- **Design Process** - Interactive timeline exploring Sprint 1 (Ideation), Sprint 2 (Prototyping), and Sprint 3 (Integration)
- **Budget (BOM)** - Bill of materials for stretch goal components (pneumatics and AI vision)
- **Code & Resources** - Python control software, AI vision system plans, and technical documentation

## Project Highlights

- **Custom 3-jaw chuck** for secure disc clamping and rotation
- **Load sensing via rotary encoder** measures motor RPM to detect disc presence
- **Raspberry Pi 5** controls all sensors, motors, and cleaning logic
- **Nema 23 stepper motor** (4.0A) with TB6600 driver for disc rotation
- **Servo motor** actuates brush with pre-programmed cleaning patterns
- **Stretch goals**: Pneumatic air system, AI vision detection, touchscreen GUI

## Tech Stack

- **Tailwind CSS** - Styling
